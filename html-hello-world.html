//for the atom git exercise Hello world

A program to greet the planet.

This page is an attempt to greet everyone on Earth using our various human languages. We're interested to see how long our list can become.

We'll start with languages that the Moringa staff could offer without using Google:

  • English: Hello World!
  • Swahili: Hujambo ulimwengu!
  • Japanese: Konnichiwa, sekai!
  • French: Bonjour monde!
  • Sheng: Niaji wasee!
  • Chichewa: Mulibwanji!
  • Spanish: Hola! Mi amigo!
